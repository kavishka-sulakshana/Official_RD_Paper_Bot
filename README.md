## PIP Dependancy Libraries

`pip install <...>`

-   python-telegram-bot
-   requests
-   python-dotenv
-   firebase-admin

### Create Virtual Environment

-   `python -m venv myenv`

### Activate the virtual Environment

-   `myenv\Scripts\activate`

### Deactivate the virtual Environment

-   `deactivate`

### Run the Bot

-   `python bot.py`
